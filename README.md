# Smart Finance Assistant: Savings Coach

## Project Overview
Savings Coach is a Smart Finance Assistant that helps users plan and stay motivated toward their savings goals. It calculates how much users need to save monthly and weekly, gives encouraging savings advice, retrieves savings tips from a small knowledge base, and displays a savings progress chart using a Gradio interface.

## Features
- Savings goal calculator
- Weekly and monthly savings breakdown
- Savings Coach chatbot
- RAG-based savings advice notes
- Savings progress chart
- Gradio user interface
- Testing section with normal, edge, and error cases

## Sample Input
Savings Goal: 5000  
Current Savings: 1000  
Months: 8  

## Sample Output
Remaining Amount: 4000  
Monthly Savings Needed: 500  
Weekly Savings Needed: 125  

## How to Run
1. Open the notebook in Google Colab.
2. Run all cells from top to bottom.
3. Launch the Gradio app.
4. Use the calculator, chart, and chatbot tabs.
